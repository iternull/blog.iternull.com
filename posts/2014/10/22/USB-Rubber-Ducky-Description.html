USB Rubber Ducky 说明介绍

2014-10-22

USB Rubber Ducky是一个可编程的USB设备
插上后识别出来的是键盘设备（不保证100%都能识别出来）
你可以用它插到其他设备上自动执行你写好的代码

硬件参数：

  1. 快速的60MHZ 32位处理器
  2. 便捷的Type A USB连接器
  3. 可通过Micro SD卡扩展内存
  4. 隐藏在不起眼的外壳中
  5. 内置载荷重按钮（为何是内置的！按个按钮还不能套壳）

使用方法分本地编译和在线生成

本地编译：

  1. 在系统上安装JAVA
  2. 创建一个exploit.txt的文件，写入代码，语法详见WiKi
  3. 编译你写好的代码

参数解释：

  • -jar 指定你用的编译器
  • -i 输入文件，你写的代码
  • -o 输出文件，最后放到SD卡里执行的程序
  1. 把inject.bin复制到SD卡，在把SD卡查到USB Rubber Ducky上
  2. 把USB Rubber Ducky查到要执行的设备上

在线生成：

  1. 打开Duck Toolkit
  2. 创建 生成你的代码和程序文件
  3. 把inject.bin复制到SD卡，在把SD卡查到USB Rubber Ducky上
  4. 把USB Rubber Ducky查到要执行的设备上

注意事项：

  1. USB Rubber Ducky不是读卡器，不能当作U盘使用
  2. USB Rubber Ducky不要乱插，万一里面放了病毒代码，后果你自己可以想象的
  3. 对于输入法可能有冲突，你要输入的是英文，结果却变成中文了，编写代码的要注意到
  4. 经常调试程序的时候一般很少把壳装上，插拔的时候要小心，别把板子弄断了
