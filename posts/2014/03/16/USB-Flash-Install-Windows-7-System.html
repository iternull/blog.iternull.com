U盘安装Windows 7系统

2014-03-16

1、下载系统镜和像工具

Windows 7 32位 镜像

Windows 7 64位 镜像

软媒魔方 软碟通

2、刻录镜像文件到U盘

刻录工具推荐选择软碟通或软媒魔方的U盘启动工具来刻录系统镜像到U盘
刻录前需要格式化U盘

3、重启进入BIOS设置USB启动

重启按Del键或F2，F12进入BIOS（具体按键依据本机型号网上自己搜索）
进入BIOS以后按左右键选择boot选项，再按上下键和-/+设置USB设备最先启动，
F10保存退出自动重启进入安装界面,
UEFI的有些引导可能是图形界面的，操作自己看一下都会懂，只要设置USB启动就行了

4、安装系统

只要按照他的引导直接下一步就可以了，分区的时候自己选择，建议全格硬盘重新分区系统盘100GB

5、安装驱动和更新

安装驱动可以使用驱动精灵带网卡版，一定要在装系统前事先下载好放到U盘里
否则装完系统以后你就只能用电脑下载了，
刚装好的系统一般是无法联网的，Ghost除外。
最好是安装系统前先用驱动精灵备份一下驱动，以免有些电脑不兼容可能出现蓝屏
系统更新，在“开始菜单” “控制面板系统和安全Windows Update”目录里检测更新并安装

  • 强烈不建议所以某卫士，某管家，或其他的第三方工具来打补丁！

6、系统激活

在计算机的属性里可以看到最下面提示试用30天，这说明你还没激活。
激活可以使用key，不过微软官方已经停止销售了Windows 7系统了，
我想在天朝普通人也不会去买正版的毕竟好几千块钱的，
所以要激活只有两个办法：

  1. 网上找可以用的key
  2. 用工具激活

建议使用nt6 oem loader v3.3.1.zip
激活后在计算机属性里会显示Windows以激活 右下角的图片显示正版授权

  • 除了向微软付钱购买的正版以外其他方法获取的都是盗版！
